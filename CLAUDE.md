# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指导。

## 项目概述

这是一个**Claude Skills 创意写作助手项目**，专注于《诛仙》风格的中文仙侠/奇幻文学创作。项目名称"溟濛记"体现了其作为写作助手的定位。

**重要提示：** 这不是传统的软件开发项目。没有构建系统、包管理器、测试框架或编译过程。该 skill 完全在 Claude 环境中运行。

## 项目结构

```
.claude/skills/写作助手/
├── SKILL.md              # Skill 主定义和写作指南
└── references/
    └── 范文.md           # 参考资料（《诛仙》第四章节选）
```

## 使用方式

当用户调用 `写作助手` skill 时：
1. 首先阅读并学习 `references/` 目录中的所有参考资料
2. 遵循 SKILL.md 中定义的写作指南
3. 将生成的创作内容输出到项目根目录

## 核心写作技巧（来自 SKILL.md）

该 skill 定义了《诛仙》风格叙事的五大核心技巧：

### 1. 有限视角叙事
- 始终通过主角（尤其是孩童主角）的感官呈现事件
- 使用"从未见过/听过"句式强调事件的非常态性
- 将宏大场景转化为身体反应（心跳、颤抖、窒息感）

### 2. 文白张力控制
- **叙述层**：现代白话为主，嵌入四字短语和文言词汇
- **对话层**：按身份区分语体（高僧文雅、反派粗厉、孩童口语）
- **口诀**："七分白，三分文，关键处用雅言点睛"

### 3. 对比艺术构建
- 场景对比：血污废墟 vs 洁净仙境
- 情绪反衬：慈悲执念 vs 阴狠贪婪
- 节奏切换：电光石火斗法 vs 绵长沉重独白

### 4. 含蓄抒情技法
公式：**强烈情感 = 细微身体反应 + 简短对话/静默 + 环境呼应**

示例：
- 悲伤："眼泪悄无声息滑落" + 默默对视 + 远方鸟鸣
- 恐惧："心脏猛跳" + "我要死了吗？" + 凌厉风声

### 5. 节奏引擎设计
模式：**平缓日常 → 危机爆发 → 情绪余震 → 新世界展开**

- **爆发段**：密集动词、短句、感官轰炸
- **余震段**：内心独白、回忆、哲学思考
- **展开段**：通过"陌生化"视角展示新环境

## 语言风格指南

- **主要语言**：中文
- **风格**：具有纯正《诛仙》韵味的仙侠/奇幻
- **目标**：减少 AI 写作痕迹，使输出更像真人创作
- **技巧**：使用长短句交替、行文流畅、节奏得当

## 适用场景

根据 SKILL.md，在以下情况使用：
- 创作古典仙侠、武侠或东方奇幻作品
- 需要营造强烈画面感与命运悲剧色彩
- 追求"文白相间、雅俗共济"的语言风格
- 需要通过有限视角增强读者代入感
