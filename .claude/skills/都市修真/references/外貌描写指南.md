# 都市修真小说 — 外貌描写指南

---

## 一、核心公式

**总结定调 + 2-3个特征快扫 + 锚点稍细**

这是所有外貌描写的基本结构。不管静态、动态、什么景别，都可以用这个公式。

**定调要敢说**——好看就说好看，身材好就说身材好，不要绕弯子。

> 那女孩猛然回头，皮肤白皙，鼻梁挺翘，一双眼睛黑亮得像含着光, 容貌甚美.

> 她弯腰放酒，领口垂下去一截。直起身才看清脸——脸也好看，五官偏冷，嘴唇薄，一双丹凤眼不带什么表情。

拆解：
- **定调**："脸也好看" "容貌甚美"——一句话让读者知道这个人好看，简洁直白
- **快扫**："皮肤白皙，鼻梁挺翘"——两笔带过，不展开
- **锚点**："一双眼睛黑亮得像含着光""丹凤眼不带什么表情"——最后一个特征多给一点笔墨，让读者记住

**字数参考**：
- 核心女主首次出场：30-80字
- 其他女主/重要女性角色：15-40字
- 配角/龙套：10字以内（"寸头的那个"、"戴金链子的"）
- 男性角色：一两个特征 + 一个气质词，不超过40字

## 二、六项核心技法

> 核心公式解决"写什么结构"，这六项解决"怎么写才好"。任何外貌描写，不管什么景别、什么出场状态，都应贯穿这些技法。

### 1. 锚定焦点细节：特征要传神，不是传真

好的锚点同时传达**外貌和性格/状态/故事**。可以直白，但不要空洞。

```
❌ "她的眼睛很大。"
   —— 空洞

✅ "一双丹凤眼，眼神清冷，带着股疏离。"
   —— 外貌（丹凤眼）+ 气质（清冷、疏离）

✅ "眼角有道浅纹，不知道是笑出来的还是熬出来的。"
   —— 外貌（浅纹）+ 故事（生活不容易）

✅ "脸是真好看，五官偏冷艳，鼻梁高，嘴唇薄。"
   —— 直白定调 + 特征快扫
```

**核心原则**：特征要有画面感，能透露性格或状态。快扫特征（"鼻梁高""皮肤白"）直接写就行，锚点多给一点笔墨。

**写观感，不写判断**：描写冷、美、飒等气质时，用观感词（清冷、疏离、淡、凌厉、慵懒），不要替读者下结论（"不像好相处的人""像欠了她钱"）。前者是你看到了什么，后者是你替读者想好了。

| 写观感 ✅ | 写判断 ❌ |
|----------|----------|
| 眼神清冷 | 不像好相处的人 |
| 带着股疏离 | 像欠了她钱 |
| 神情淡淡的 | 让人不自觉退半步 |
| 目光不带什么温度 | 像是什么都见过了 |
| 透着一股生人勿近的劲儿 | 不太好惹 |

### 2. 动作带出外貌：让行为"泄露"特征

可以直接写特征，但如果能让动作自然暴露，效果更好——一笔同时给外貌和性格。

> "她整理吧台上的酒杯，手很稳，动作干净利落——指节分明，是干活的手，不是养的。"

> "她低头看书的时候，头发滑下来挡住半边脸，她随手别到耳后，过一会儿又滑下来。"

> "她弯腰放酒的时候，领口垂下去一截，锁骨下面一片白。她自己毫不在意。"

> "她站起来比旁边两个男生都高半个头。走路手插口袋，步子不快，但别人会自觉让路。"

**不是说不能直接写**。"腿很长""皮肤白""腰细"这些直接写完全没问题。只是如果能挂在一个动作上，就多了一层性格信息。能挂就挂，没必要硬挂。

### 3. 以物衬人：衣着是无声的台词

衣着不只是"好看不好看"，它同时说明三件事：**经济水平、性格、当前状态**。

| 角色 | 衣着写法 | 暗示了什么 |
|------|---------|----------|
| 林青雅 | 校服干干净净，不化妆，发尾自然弯 | 天生丽质，不需要打扮也好看 |
| 顾书颜 | 穿衣不花哨但每一件都贵得安静，偏分露出耳钉 | 有钱且有品味，低调的精致 |
| 秦晓曼 | 化了妆是酒吧最扎眼的，卸了妆素面朝天 | 上班和回家判若两人 |
| 赵晴 | 便装随意像大学生 vs 制服马尾干练 | 同一个人的两面——长相和行动力的反差 |
| 小白 | 染黑的头发发根泛白，棕色美瞳 | 在掩饰什么，但掩饰得不太成功 |

**关键**：不要写"她穿了一件XX"然后没了。穿着细节要暗示身份和性格——"校服永远整洁"说明这人讲究，"袖子卷到手肘"说明这人干活不惜力。

### 4. 写精气神，不只写好看

最常见的陷阱：所有女性角色的描写都是"她好漂亮，男主看呆了"。

好的描写不只写她**长什么样**，更写她**怎么活在这个世界里**——她的精气神、她的存在感、她做事的方式。

```
❌ 纯凝视：
   "他看着她的侧脸，白皙的肌肤在灯光下格外好看。"
   —— 只有"她被看"

✅ 精气神：
   "她整理完最后一杯酒，擦了擦手，目光在酒吧扫了一圈——
    那种把所有事都攥在手心里的松弛感，和她脸上的淡妆完全不匹配。"
   —— 有"她是谁"
```

```
❌ "林青雅长得很漂亮，笑起来很好看。"

✅ "林青雅把整理好的笔记放在他桌角，轻声说：'这周的重点我都标了。'
    谢知远抬头看她——她笑了一下，眼睛弯弯的，像在说'别客气'。"
```

**核心区别**：
- 凝视写的是：观察者看到了什么（她是被看的对象）
- 精气神写的是：这个人怎么存在的（她是有主体性的人）

两者可以结合，但不能只有凝视。每个女性角色都应该有自己"做事的样子"——陪酒、办案、谈判、打拳、看书——这些才是她真正的魅力来源。外貌是锦上添花，不是全部。

### 5. 面貌+气质=完整印象

单纯写脸不够。读者记住一个人，靠的不是五官参数，而是**面貌和气质的叠加**。

| 光写面貌 | 面貌+气质 |
|---------|----------|
| "五官精致" | "五官其实不算多出挑，但那种从容的劲儿让她整个人都不一样" |
| "长得很帅" | "不算帅，但眼神稳，不像这个年纪的人" |
| "她很美" | "不化妆也好看，笑起来嘴角弯弯的，让人不自觉跟着心情好" |

气质怎么写？绑定到**体态和动作**上：
- 行走如风 = 干练果断
- 慢条斯理 = 从容/优雅
- 站在那里不说话也让人有压迫感 = 气场
- 说话的时候手上在做别的事 = 老练/忙碌

### 6. 情境化描写：同一个人，不同场景不同样子

不要一次写完一个人的全部外貌。同一个人在不同场景下，读者看到的应该是不同的面。

**秦晓曼在不同场景：**

| 场景 | 样子 | 新细节 |
|------|------|-------|
| 酒吧工作 | 妆容精致，露肩短裙，笑得甜但眼神冷 | 职业化的假面 |
| 回家照顾母亲 | 棉拖鞋棉睡衣，蹲在地上给母亲擦身，眉眼间的冷全化了 | 真实温柔的一面 |
| 和谢知远独处 | 素颜，声音比酒吧里轻，偶尔发呆 | 卸下防备的一面 |

**赵晴在不同场景：**

| 场景 | 样子 | 新细节 |
|------|------|-------|
| 林家聚餐（便装） | 头发散着，穿得随意，看着像大学生 | 清秀文静 |
| 执行任务（制服） | 马尾扎紧，跑起来动作利索，眼神锐利 | 和便装判若两人 |
| 受伤/疲惫后 | 头发散了一半，但眼神没散 | 倔强的一面 |

**关键**：每个场景只给一两笔新细节，不重复已经写过的。读者对角色的印象就这样一层一层丰富起来。这比一次性铺完所有外貌信息效果好得多。

## 三、描写模式

### 动态描写

人物在运动——走、跑、回头、推门。外貌伴随动作自然释放。

> 走廊尽头过来一个女生，长发垂在肩上，发尾微弯。走到跟前停下，五官精致，一双眼睛又大又清亮。

### 静态描写

人物静止——坐着、站着、低头做事。视线扫过去，被什么吸引就写什么。

> 吧台后面站着个女人，低头擦杯子。低马尾，淡妆，底子好，五官耐看，眼角有道浅纹。

### 混合描写（最常用）

静止的人突然有了动作——先静态打底，一个动作触发细节。

> 她一直低头看手机。忽然抬眼看过来——容貌精致，一双眼尾微挑的眼睛，带着点说不清的意味。

## 四、景别参考

| 景别 | 看到什么 | 用在哪 |
|------|---------|--------|
| **远景** | 身形、走路姿态、衣着轮廓 | 人物还没走近，先给模糊印象 |
| **中景** | 衣着细节、发型、大致面部 | 已经能看清穿什么、什么发型 |
| **近景** | 五官中某一个、表情 | 走近了或面对面，锚点在这里给 |
| **特写** | 极致细节——纹理、肤质、微表情 | 亲密场景、重逢、心动瞬间。首次出场慎用 |

景别推进通常是 **远→近**，符合自然观察。但不是每次都要走完全程——根据出场状态选择。

## 五、不同出场状态的写法

### 远处走来

远景给轮廓，走近后总结定调 + 锚点。

> 走廊尽头走过来一个女生，个子高，穿白T恤牛仔裤。走近了才看清——长得漂亮，笑起来有颗小虎牙。

### 已在场（静止）

中景扫一眼打底，等一个动作触发近景。

> 三排外坐着个穿黑西装的女人，锁骨发，左手腕上一块男款手表。她举牌的时候侧了下头——五官偏冷，嘴角微挑，像在打什么算盘。

### 突然近距离照面

跳过远中景，直接近景，冲击力最强。

> 拐角差点撞上个人。帽檐一抬，露出半张脸——皮肤很好，一双眼睛不大，但有点空，像在看什么远处的东西。

### 特殊状态（化形/昏迷/受伤）

观察时间充裕，可以多写一两笔，但仍以总结 + 锚点为主。

> 阵法中央蜷着个女孩。一头白发散落满地，皮肤白得不像话。她缓缓抬头——容貌清丽，一双浅琥珀色的眼睛，眼尾微挑，没有焦距，像是还没弄明白自己在哪。

### 再次出场

不重复描写，一笔锚点唤醒记忆。

> 她把马尾甩到肩后。

> 腕上那块男款手表晃了一下。

> 她抬了下眼皮，眼角那道浅纹跟着动了动。

## 六、身材描写

> **核心态度：直白但有画面。** 身材好就说身材好，腿长就写腿长，该写的不要绕。但要挂在动作或衣着上，写出画面感，不能只扔一句"身材火辣"。

### 核心原则：直白 ≠ 廉价

```
❌ "她身材火辣，凹凸有致。"
   —— 套话，没画面，谁都能用

✅ "吊带裙勒出一条流畅的轮廓，腰细得过分，但胸前撑得很满，走起来晃得不明显。"
   —— 直白+有画面+有动作

❌ "她身高一米七，比例很好，腿很长。"
   —— 体检表

✅ "裙子很短，露出两条又直又长的腿。"
   —— 直白+简洁，一句话就够了
```

**简单说**：你可以写"腿很长""胸很大""腰很细"——只要它出现在一个具体的画面里，不是干巴巴的参数。

### 三种写法

**1. 直写——挂在动作或衣着上**

最直白的写法。身材信息直接给，但嵌在动作或衣着中，不是单独陈列。

> "她弯腰放酒，领口垂下去一截，锁骨下面一片白晃晃的。"

> "吊带裙很短，堪堪盖住大腿根，露出两条又直又长的腿。腰细得过分，但胸前撑得很满。"

> "她站起来，旁边那男生的视线刚到她下巴。腿长，腰也细，穿什么都是衣架子。"

> "她把外套搭在椅背上，露出一截手臂——瘦，但不是弱的那种瘦，是有劲儿的。"

**2. 衣着勾勒——让衣服替你说**

衣服松了紧了、绷了垮了，身材信息就有了。

> "那件oversize卫衣穿在她身上空荡荡的，显得整个人更小了一号。"

> "西装是定制的，肩线刚好，一点余量都没有。腰那一截收得很紧。"

> "工作服绷了一下，她没注意到——或者说，懒得注意。"

> "裙子紧身，身材藏不住。她往那一坐，腿一叠，神情很淡，目光带着股疏离。"

**3. 触感——近距离肢体接触时**

不是看到的，是碰到时感受到的。适合暧昧场景。

> "谢知远扶住她胳膊的时候才发现，她比看上去轻很多——骨架小。"

> "她靠过来，肩膀抵在他手臂上，很软，带着点体温。"

> "他一把拉住她的手腕，细得一握就满了。"

### 各角色身材差异

身材要和角色的身份、经历、性格匹配。不是所有人都"身材火辣"——有的人的好看是健康有力量感的，有的是纤细的，有的是成熟的。

| 角色 | 身材 | 怎么写 | 为什么 |
|------|------|-------|-------|
| **林青雅** | 纤细匀称，不瘦弱但也不是运动型 | "校服穿在她身上刚好""坐在旁边的时候，肩膀窄窄的" | 校花型，养得好 |
| **小白** | 十五六岁少女体型，纤细瘦小 | "oversize卫衣空荡荡的""看上去一阵风能吹跑""骨架小得可怜" | 化形不久 |
| **顾书颜** | 高挑纤瘦，天生衣架子 | "穿什么都像在走秀""站起来的时候旁边男生只到她肩膀" | 富家千金，形体管理到位 |
| **苏珊** | 匀称利落，不高不矮 | "穿正装好看穿便装也好看""不是那种需要挑衣服的体型" | 海归精英，生活规律 |
| **陆雨桐** | 偏瘦娇小，比想象中小一号 | "电视上看着高挑，站面前才发现踩了恨天高""摘了帽子比想象中小一号" | 艺人，镜头放大 |
| **赵晴** | 结实不壮，线条紧实 | "跑起来动作利索""看着文静，胳膊上的力道出乎意料" | 警校出身 |
| **秦晓曼** | 身材好，腰细，胸前撑得满，腿直且长 | "露肩短裙穿在身上，腰线收得紧，胸前撑得满""卸了妆回家完全是另一个人" | 天生底子好 |

### 禁区与尺度

| 始终禁用 | 为什么 |
|---------|-------|
| 凹凸有致、前凸后翘、魔鬼身材、身材火辣 | 套话，没画面。直接写"腿长""胸大""腰细" |
| 所有女主都"身材完美" | 每个角色身材类型不同，要有辨识度 |

| 看场景使用 | 规则 |
|-----------|------|
| 具体身体部位详写（胸、臀、腿的轮廓/动态） | 酒吧、夜店、暧昧、换装等场景**可以放开写**，正常校园/职场场景**克制** |
| 连续多句描写身体 | 特定场景（陪酒女出场、暧昧桥段）**允许整段写身材**；日常场景一两句带过 |
| 感官化细节（晃动、勒痕、沟） | 场景氛围到了就写，不用回避。但不要每个女角色出场都这么写 |

**判断标准**：这段身体描写是服务于场景氛围和人物塑造，还是单纯在"看"？如果角色是陪酒女、场景在酒吧，详细写身材就是在建立场景和人物身份，不是凝视。如果角色是同事、场景在办公室，连续三句写胸就是凝视。

### 时机

| 时机 | 写什么 |
|------|-------|
| **首次出场** | 一两笔带过整体——"个子高""腿长""瘦"，嵌在动作里 |
| **换装/暴露场景** | 衣着变化自然暴露——裙子短了、领口低了、袖子卷了 |
| **近距离/肢体接触** | 触感——轻重、骨感、体温、柔软 |
| **运动/战斗场景** | 力量感、灵活度、修真前后体质变化 |

## 七、不同类型角色的描写差异

不同风格的女性角色，定调用词、特征选择、锚点方向都应该不同。

| 类型 | 定调方向 | 特征偏好 | 锚点示例 |
|------|---------|---------|---------|
| **清纯甜美型**（林青雅） | 清纯、甜、校花 | 黑长发、大眼睛、甜笑 | "一双眼睛又大又清亮，笑起来嘴角弯弯的" |
| **妖冶型**（小白） | 清丽、不像话 | 白发、异瞳、皮肤过白 | "赤红色的眸子，眼尾微挑，天生带着点说不清的意味" |
| **精致冷感型**（顾书颜） | 精致、冷、距离感 | 波浪偏分、狐狸眼、贵气穿搭 | "笑起来嘴角上扬的弧度刚刚好，像在打量你值不值得她多看一眼" |
| **知性干练型**（苏珊） | 知性、干练、耐看 | 锁骨短发、银框眼镜 | "不笑的时候像在想什么重要的事，笑起来眼角弯弯的" |
| **反差型**（陆雨桐） | 反差、甜、疏离 | 口罩鸭舌帽、梨涡 | "不笑的时候五官偏冷，一笑起来两颗梨涡——整个人都不一样了" |
| **清秀反差型**（赵晴） | 清秀、显小、干练 | 翘鼻头、显小长相、跑起来利索 | "长得像大学生，跑起来动作利索得吓人" |
| **冷艳性感型**（秦晓曼） | 冷艳、性感、外热内冷 | 丹凤眼眼尾上挑、眼神清冷带疏离、身材好、职业微笑 | "眼神清冷，带着股疏离""笑是练出来的，不到心里" |

**关键**：每个角色的描写要有辨识度。把名字遮住，读者也能分辨是谁。

## 八、后续补充描写的时机

首次出场只给总结 + 锚点。后续在自然时机补充新细节：

| 时机 | 补充什么 |
|------|---------|
| 换装/换场景 | 衣着反差（情境化描写的核心时机） |
| 近距离接触 | 皮肤质感、气味、手的细节 |
| 不同光线 | 阳光/灯光/月光下的不同感觉 |
| 情绪变化 | 笑/哭/生气时的特征 |
| 工作/专注状态 | 精气神——她做事的样子才是真正的魅力（技法4） |
| 男主视角转变 | 从"还行"到"好看"到"移不开眼" |
| 亲密时刻 | 呼吸、体温、睫毛 |

## 九、男性角色

气质胜于容貌，行为重于外表。一两个特征 + 一个气质词，不超过40字。

> 身形单薄，但脊背挺得直。容貌清秀，不算扎眼，但眼神沉得不像这个年纪。

## 十、避免清单

**禁的是套话和空话，不是直白。**

| 禁用 | 为什么 | 替代 |
|------|-------|------|
| 凹凸有致、前凸后翘、魔鬼身材、身材火辣 | 套话，没画面 | 直接写——"腿长""胸大""腰细"，挂在画面上 |
| 沉鱼落雁、倾国倾城、明眸皓齿 | 成语堆砌 | 直说——"脸是真好看""长得漂亮" |
| "她有着一头...一双...一个..." | AI句式 | 嵌进动作——"她抬头，眼睛很亮" |
| 不禁、竟然、居然、这一刻、映入眼帘 | AI高频词 | 删掉 |
| 仙女下凡、天使降临 | 俗套 | 删掉 |
| 惊为天人、美得不可方物 | 过度反应 | "多看了一眼""愣了一下" |
| 整个人散发着XX的气质 | 空话 | 删掉 |
| "不像好相处的人""像欠了她钱""让人退半步" | 替读者下判断 | 用观感词——"眼神清冷""神情淡淡""带着疏离" |
| 从头发逐条写到鞋子 | 流水账 | 抓1-2个点 |
| 三围数据、具体身高体重 | 体检表 | 用画面暗示——"腰一只手就能圈住""比旁边男生高半个头" |

**允许的直白写法**（这些不是禁区）：
- "长得漂亮""脸是真好看""身材是真好"——直说没问题
- "腿很长""腰很细""胸前撑得满""屁股翘"——具体部位可以写，要有画面
- "皮肤白""鼻梁高""嘴唇薄"——简洁的特征快扫，两三笔带过
- "领口低，挤出一道沟""胸口跟着晃了一下"——场景合适就可以写，不用回避

## 九大、统一原则

- **写观感，不写判断**：气质用观感词——"清冷""疏离""淡""慵懒"；不替读者下结论——不写"不好相处""像欠了她钱""让人退半步"
- **特征抓辨识度**：每个角色的锚点必须不同，不能所有人都"眼睛很亮"
- **特征要传神**：锚点带性格/状态，但快扫特征直接写就行，不用每个都"传神"（技法1）
- **动作带外貌**：能挂在动作上就挂，没必要硬挂（技法2）
- **衣着是台词**：穿着暗示经济水平+性格+当前状态（技法3）
- **写精气神**：描写她做事的样子，不只描写她被看的样子（技法4）
- **面貌+气质**：把外形和体态/气场绑在一起写（技法5）
- **情境化**：同一个人在不同场景写不同的面，一层一层丰富（技法6）
- **外貌与性格统一**：懒散的人穿拖鞋，利落的人头发从不乱
- **锚点复用**：再次出场提一下锚点就够，不重复全貌
- **服务剧情**：精心打扮=有事，眼圈发黑=熬夜，穿搭暗示身份
