# 都市修真小说 — 外貌描写指南

---

## 一、核心公式

**总结定调 + 2-3个特征快扫 + 锚点稍细**

这是所有外貌描写的基本结构。不管静态、动态、什么景别，都可以用这个公式。

> 那女孩猛然回头，容貌甚美，皮肤白皙，鼻梁挺翘，一双眼睛黑亮得像含着光。

拆解：
- **定调**："容貌甚美"——一句话让读者知道这个人好看，不绕弯子
- **快扫**："皮肤白皙，鼻梁挺翘"——两笔带过，不展开
- **锚点**："一双眼睛黑亮得像含着光"——最后一个特征多给一点笔墨，用比喻或具象细节让读者记住

**字数参考**：
- 核心女主首次出场：30-80字
- 其他女主/重要女性角色：15-40字
- 配角/龙套：10字以内（"寸头的那个"、"戴金链子的"）
- 男性角色：一两个特征 + 一个气质词，不超过40字

## 二、描写模式

### 动态描写

人物在运动——走、跑、回头、推门。外貌伴随动作自然释放。

> 走廊尽头过来一个女生，步子快，马尾甩来甩去。走到跟前停下，五官明朗，眼睛很亮。

### 静态描写

人物静止——坐着、站着、低头做事。视线扫过去，被什么吸引就写什么。

> 吧台后面站着个女人，低头擦杯子。低马尾，淡妆，底子好，五官耐看，眼角有道浅纹。

### 混合描写（最常用）

静止的人突然有了动作——先静态打底，一个动作触发细节。

> 她一直低头看手机。忽然抬眼看过来——容貌精致，一双眼尾微挑的眼睛，带着点说不清的意味。

## 三、景别参考

| 景别 | 看到什么 | 用在哪 |
|------|---------|--------|
| **远景** | 身形、走路姿态、衣着轮廓 | 人物还没走近，先给模糊印象 |
| **中景** | 衣着细节、发型、大致面部 | 已经能看清穿什么、什么发型 |
| **近景** | 五官中某一个、表情 | 走近了或面对面，锚点在这里给 |
| **特写** | 极致细节——纹理、肤质、微表情 | 亲密场景、重逢、心动瞬间。首次出场慎用 |

景别推进通常是 **远→近**，符合自然观察。但不是每次都要走完全程——根据出场状态选择。

## 四、不同出场状态的写法

### 远处走来

远景给轮廓，走近后总结定调 + 锚点。

> 走廊尽头走过来一个女生，个子高，穿白T恤牛仔裤。走近了才看清——长得漂亮，笑起来有颗小虎牙。

### 已在场（静止）

中景扫一眼打底，等一个动作触发近景。

> 三排外坐着个穿黑西装的女人，锁骨发，左手腕上一块男款手表。她举牌的时候侧了下头——五官偏冷，嘴角微挑，像在打什么算盘。

### 突然近距离照面

跳过远中景，直接近景，冲击力最强。

> 拐角差点撞上个人。帽檐一抬，露出半张脸——皮肤很好，一双眼睛不大，但有点空，像在看什么远处的东西。

### 特殊状态（化形/昏迷/受伤）

观察时间充裕，可以多写一两笔，但仍以总结 + 锚点为主。

> 阵法中央蜷着个女孩。一头白发散落满地，皮肤白得不像话。她缓缓抬头——容貌清丽，一双浅琥珀色的眼睛，眼尾微挑，没有焦距，像是还没弄明白自己在哪。

### 再次出场

不重复描写，一笔锚点唤醒记忆。

> 她把马尾甩到肩后。

> 腕上那块男款手表晃了一下。

> 她抬了下眼皮，眼角那道浅纹跟着动了动。

## 五、不同类型角色的描写差异

不同风格的女性角色，定调用词、特征选择、锚点方向都应该不同。

| 类型 | 定调方向 | 特征偏好 | 锚点示例 |
|------|---------|---------|---------|
| **明朗型**（林青雅） | 明朗、干净、好看 | 马尾、笑容、眼神亮 | "眼睛很亮，不是化妆画出来的那种亮" |
| **妖冶型**（小白） | 清丽、不像话 | 白发、异瞳、皮肤过白 | "浅琥珀色的眼睛，眼尾微挑" |
| **冷艳型**（顾书颜） | 冷、精致 | 利落发型、冷色穿搭、配饰 | "嘴角微挑，像在打什么算盘" |
| **知性型**（苏珊） | 从容、干练 | 西装配休闲、珍珠耳钉 | "不笑的时候像不好说话的人，笑起来和外表完全不搭" |
| **甜美型**（陆雨桐） | 甜、反差 | 棒球帽、酒窝 | "笑起来酒窝很深，不笑的时候眼神有点空" |
| **英气型**（赵晴） | 飒、反差 | 便装vs制服、虎牙 | "便装像大学生，穿上制服走路带风" |
| **风情型**（秦晓曼） | 耐看、有故事 | 淡妆、手稳、眼角细纹 | "眼角有道浅纹，是笑出来的，也是熬出来的" |

**关键**：每个角色的描写要有辨识度。把名字遮住，读者也能分辨是谁。

## 六、后续补充描写的时机

首次出场只给总结 + 锚点。后续在自然时机补充新细节：

| 时机 | 补充什么 |
|------|---------|
| 换装/换场景 | 衣着反差 |
| 近距离接触 | 皮肤质感、气味、手的细节 |
| 不同光线 | 阳光/灯光/月光下的不同感觉 |
| 情绪变化 | 笑/哭/生气时的特征 |
| 男主视角转变 | 从"还行"到"好看"到"移不开眼" |
| 亲密时刻 | 呼吸、体温、睫毛 |

---

## 七、男性角色

气质胜于容貌，行为重于外表。一两个特征 + 一个气质词，不超过40字。

> 一米七八，偏瘦，骨架撑得住。不算帅，但眼神稳，不像这个年纪的人。

---

## 八、避免清单

| 禁用 | 替代 |
|------|------|
| "她有着一头乌黑的长发，一双明亮的大眼睛，白皙的肌肤" | 总结定调 + 锚点 |
| 不禁、竟然、居然、不由得、这一刻、映入眼帘 | 删掉，直接写 |
| 沉鱼落雁、倾国倾城、明眸皓齿 | 用具体的——"眼睛黑亮""鼻梁挺翘" |
| 仙女下凡、天使降临、画中走出来的人 | 删掉 |
| 惊为天人、美得不可方物、看呆了 | "多看了一眼""愣了一下" |
| 整个人散发着XX的气质 | 删掉，空话 |
| 从头发逐条写到鞋子 | 只写被吸引的1-2个点 |

---

## 九、统一原则

- **总结可以直说**：容貌甚美、五官精致、长得漂亮——都可以用，简洁明了
- **特征抓辨识度**：每个角色的锚点必须不同，不能所有人都"眼睛很亮"
- **外貌与性格统一**：懒散的人穿拖鞋，利落的人头发从不乱
- **锚点复用**：再次出场提一下锚点就够，不重复全貌
- **服务剧情**：精心打扮=有事，眼圈发黑=熬夜，穿搭暗示身份
