## 故事背景

```
盘古开天，混沌初分。天地精气交感，孕化万灵。

太古之时，妖兽横行。强者移山倒海，斗转星移；弱者潜藏幻化，神通莫测。彼时万族相争，杀伐不断，天地为之一片混沌。史称太古之乱。

岁月悠悠，不知几千百载，龙神横空出世。以无上法力镇压四方妖兽，复天地以清明。世遂太平。

龙神纪三万六千年，龙神以大神通开辟异界，尽放逐妖兽于其中。后世称之为妖界。

妖兽与龙族，皆寿逾万年，生命力强横异常。龙神坐化后，龙族世代守护天地，传承不辍。

龙族圣女,娲，仿龙神之法，抟土造人。人族由此登上历史舞台。

人族虽天赋异禀，却寿不过百，生命脆弱如朝露。人族为龙族附庸。

后世龙皇，效仿龙神开辟龙界，率全族遁入其中。龙族离开后，人族遂得龙族传承之修行法门，渐渐崛起于天地之间。

悠悠岁月，人族创立无数修行门派。其中四派威震天下：东原武宗、 男孩观音宗、西山碧落山、北冥龙鲲。

然有正便有邪，门派分善恶。正道之外，还有魔道。魔道主要有三大门派: 五域鬼窟，天魔宗， 三妖门。

太古之后，又一轮风云，即将开启……

```

## 人物关系:
```
男主角: 叶涯，南方小城，城主之子。其父亲意外得到了金色指骨， 引来各方强者窥伺。后叶家被神秘黑衣人灭门，叶涯不在家中，幸免遇难。之后和龙曦结伴闯荡江湖。
女主角: 龙曦，龙皇之女，奉龙皇之命， 通过空间缝隙意外来到人间， 寻找龙神遗蜕。在空间穿梭中受伤，流落街头。后被叶涯所救。
```
