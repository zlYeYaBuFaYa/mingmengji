# 林青雅 - 人物卡片

## 基本信息
- 年龄/身份：18岁，A市第一中学高三学生，学习委员，全校公认校花
- 外貌：一头黑长发自然垂在肩上，发尾微微弯。肤色白皙，五官精致，一双眼睛又大又清亮，看人的时候像盛着一汪水。笑起来嘴角弯弯的，甜得让人移不开视线
- 视觉锚点：清亮的大眼睛 + 黑长发 + 弯弯的甜笑
- 美感类型：清纯甜美型
- 性格关键词：善良温柔、阳光自信、偶尔迷糊
- 口头禅/习惯动作：紧张时咬下唇；找不到笔（夹在耳朵上）；被调侃时脸红到耳朵根但嘴硬否认
- 背景故事：父亲林正邦（A市刑警队大队长），母亲苏敏华（大学副教授）。家境良好，家教严格不压抑，在充满爱的家庭长大。性格开朗，遇事不轻易消沉

## 语言指纹
- 说话句式特征：语气温和但有主见，认真时会用长句讲道理，日常偏活泼
- 用词偏好：常用"你是不是……""没关系的""你别这样"；关心人时措辞自然到像在说自己的事
- 情绪表达方式：高兴时笑得眉眼弯弯；心疼时眼眶泛红但忍住不说；害羞时低头摆弄头发
- 典型对话模式：作为学委催作业语气认真但不凶；后期关心谢知远时找学委职责当借口（"反正我也要复习""你拉低班级平均分我也有责任"）；被点破后否认但脸先红了
- 语言样本：
  - "谢知远，作业。"——收作业时的标准句式，语气平静
  - "反正我也要复习，多讲一遍我自己也能记得更牢。"——找借口关心
  - "我只是怕他拉低班级平均分！"——被调侃时嘴硬
  - "你最近是不是太累了？"——试探性关心，声音比平时轻
  - "嗯！"——笑着点头，声音软软的

## 情感线
- 与男主关系阶段：他暗恋/她无感 → 他变化/她好奇 → 主动接触 → 好感萌芽 → 在意（第一卷进度）
- 空间关系：谢知远坐最后一排靠窗，在班里透明人一样的存在。林青雅是学委，每次收作业会路过他桌前
- 初次关注契机：知道他家里出了事（并非全部细节），所以从不因为成绩看不起他。作为学委的责任心+天生善良，对他比对其他差生多一份耐心
- **前期（修真前）——他暗恋她，她无感**：
  - 谢知远暗恋她——但这完全是单方面的。他坐最后排靠窗，上课睡觉，林青雅来收作业时他嬉皮笑脸糊弄两句，这就是全部交集了
  - 林青雅对谢知远的印象仅限于"班里那个老不交作业的学渣"。她善良，不看不起差生，但也不会特意关注——学委的职责是收作业记考勤，不是当保姆
  - 谢知远偶尔的嘴贫，在她看来就是差生在耍赖皮，没往心里去
- **中期（变化期）——好奇驱动**：
  - 谢知远启灵后发生变化：记忆力/理解力飙升，开始认真看书；自信心建立，嘴变得更贫，偶尔说出让人意外的话；气质从"懒散透明人"变得有棱角
  - 最直接的信号是**成绩变化**——一个万年倒数的人突然开始交作业、月考成绩飙升，这在班里是大新闻。林青雅作为学委第一个注意到
  - 林青雅开始好奇："他怎么突然开始学了？"——从"无视"变成"留意"
  - 谢知远可能做了某些让她刮目相看的事（帮她解围、替人出头等，具体事件由正文决定），打破了她对他的固有印象
  - 好奇驱动下，她开始主动和谢知远说话——以学委身份找理由（"你最近进步挺大的，要不要我帮你整理一下重点？"）。补课在**这个阶段**才合理，因为谢知远自己已经表现出了学习意愿
- **后期（好感→在意）**：
  - 接触增多后发现谢知远嘴贫有趣、关键时刻靠谱、跟表面的吊儿郎当完全不一样
  - 食堂偶尔坐他对面、走时把没喝完的牛奶留在桌上——这类行为出现在频繁接触之后，是她下意识的关心，自己都没察觉
  - 如果此时谢知远家里出了变故（负债事件），她目睹他的处境后，善良+已有的好感叠加，关心会更进一层
  - 被同学调侃"你对谢知远也太好了"时脸红否认——她自己还没意识到从"好奇"变成了"在意"
- 暧昧风格标签：慢热渐进型。从无视到好奇（因为他的变化）→ 从好奇到接触（以学委身份找理由）→ 从接触到在意（发现他跟想象的不一样）→ 在意到最后才发现已经越界了
- **核心张力**（后期显现）：刑警队长的女儿和负债家庭的儿子。这个张力在她产生真正的好感后才有意义，之前谢知远跟她的世界没有交集，根本不存在这个问题

## 剧情定位
- 出场卷/章：第一卷第2章
- 主要作用：暧昧担当/情感核心/校园线驱动
- 人物弧线：对谢知远毫无感觉的校花学委 → 被他修真后的变化引起好奇 → 接触后发现他跟想象中完全不一样 → 从"在意"到"心动" → 面对身份差距和父亲态度的考验 → 成长为有勇气追求自己选择的人
- 大学去向：考入H市大学文学院，与谢知远同校不同专业，延续暧昧线

## 经典台词
- "谢知远，作业。"——标准的学委收作业句式
- "每次都这个理由，你能不能换一个？"——无奈但不生气
- "你脑子不差。"——第一次对谢知远表达善意的关心
- "你就拿这种歪理糊弄自己吧。"——对谢知远"铁杵木杵"歪理的回应
- "你今天不一样。"——注意到谢知远心事的观察力
